GUI Pong game in Zig, using Raylib (Re-written from SDL2)

## Quick Start

### Building From Source

Requirements:

- Zig v0.11.0

```console
$ zig build run
```

### Binary Download

I have a release available with a windows binary

Linux users, feel free to try to run this with WINE
